== Description ==

### Adds ARS (Argentine peso) currency in Jigoshop ###

Please notice that [Jigoshop](http://wordpress.org/extend/plugins/jigoshop/) must be installed and active.

Works with [Jigoshop MercadoPago](http://wordpress.org/extend/plugins/jigoshop-mercadopago/).

== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Jigoshop -> Settings -> Shop and choose **Argentine peso** in **Currency** option.
